How to Play the game

Welcome to the game where you must find the flags.



In this game you are hired to be a penetration tester for Lumino. So you need to find an exploit on this site which will tell you where to go next
If you ever get stuck, look for hints hiding around. There are three different exploits to work with.
If you find the last flag, you will get a custom certificate of completion upon submitting it in this form.
Even if you don't get the final flag, please take the time to answer the questions so future developers of Lumino can make fixes, patches, and new levels that you might find interesting.
Also yes, to get the certificate you do have to give feedback. Real penetration testers have to write reports on their findings so this is just some practice.

Don't forget to have fun and if you break something, it should be fine in a day.
